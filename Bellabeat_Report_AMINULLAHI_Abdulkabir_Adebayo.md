# Bellabeat Case Study

**Author:** AMINULLAHI, Abdulkabir Adebayo

## Introduction

This project analyzes user wellness behaviors using Fitbit data
processed in Google BigQuery and visualized in Tableau.

## Tools Used

-   Google BigQuery\
-   BigQuery Standard SQL\
-   Tableau\
-   CSV Files

## Key Insights

-   15 medium-activity users\
-   14 low-activity users\
-   Sleep shows a mild positive relationship with steps\
-   Calories burned closely follow step counts

## Marketing Recommendations

-   Promote activity habit-building\
-   Emphasize sleep improvement\
-   Personalize engagement\
-   Highlight calorie-burn feedback
